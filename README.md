# Chat

A simple realtime chat website. Made with JavaScript, Socket.io and Bootstrap

### Requirements

- [Node](https://nodejs.org/en/)

### Instalation

After clone the repository, install the dependencies using:

`npm install`

### Starting the server

`npm start`
